## how to use

| command | description |
| ------ | ------ |
| Ctrl+s |	 保存 	|			
| Ctrl+q |	 退出gvim，一般先保存，再进行这个操作 | 				
| Ctrl+e |	 刷新 				|
| Ctrl+j 或 Ctrl+k 	| 打开多个编辑窗口时，上下调节窗口大小 		|		
| Ctrl+h 或 Ctrl+l |	 打开多个编辑窗口时，左右调节窗口大小 		|		
| Ctrl+n |	 打开多个文件时，在不同文件间切换，切换到下一个文件 |		
| Ctrl+p |	 打开多个文件时，在不同文件间切换，切换到上一个文件 |		
| F5 	| 添加文件头 	|			
| F6 |	 新建sv文件，自动生成部分代码 			|
| \a 	| 缩进混乱的sv文件，自动缩进 			|
